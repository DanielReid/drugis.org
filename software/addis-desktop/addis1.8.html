Release Date: 2011-07-01

We are happy to announce the release of ADDIS (Aggregate Data Drug Information System) version 1.8. This release focusses on additional data modeling to represent study design. New features include:

  • Study time structure modeling.
  • Allow missing measurements.
  • Support primary/secondary endpoints.
  • Generate summary of efficacy table according to EMA D80 template.

Work on study design representation will continue for the next release.

Running ADDIS

Extract the addis-1.8.zip ZIP archive to a local directory. Then double-click the 'addis-1.8.jar' file. Now you should see the ADDIS welcome screen, where you can choose to load an example data set, load a file, or create a new data set. If you've never used ADDIS, we recommend that you take a look at the example data set.

If double-clicking the JAR file did not launch ADDIS, you are most likely lacking the Java run-time environment. You can get this from java.com.
