Release history

Release 0.4.1

Release Date: 2009-12-08

ADDIS v0.4.1 is a maintanance release and contains no new features. Several issues are addressed by this release:

  • Corrected standard deviation in Organon studies.
  • Fixed the issue where an old domain.dat could prevent ADDIS from starting.
  • Due to a packaging mistake, ADDIS v0.4 required Java SE 6, v0.4.1 requires Java SE 5.

Release 0.4

Release Date: 2009-11-26

We are happy to announce the release of ADDIS (Aggregate Data Drug Information System) version 0.4. The goal for this release was to develop an experimental prototype that allows us to perform random effects meta-analyses of clinical trials data and show the associated forest plots.

We implemented DerSimonian and Laird random effects meta-analysis and tabular and graphical representation of data. This will provide a basis for the development of more advanced analysis techniques and eventually enable benefit-risk analysis.

Running ADDIS

Extract the addis-0.4.1.zip ZIP archive to a local directory. Then double-click the 'addis.jar' file. Now you should already have ADDIS 0.4 running with some provided example data. Verify this by comparing your screen with the screenshots below.

If double-clicking the JAR file did not launch ADDIS, you are most likely lacking the Java run-time environment. You can get this from java.com.
