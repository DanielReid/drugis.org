Release Date: 2009-06-30

Download ADDIS v0.2

We are happy to announce the release of ADDIS (Aggregate Data Drug Information System) version 0.2. The goal for this release was to develop an experimental prototype that allows us to perform simple meta-analyses of clinical trials data as well as benefit risk assessment based on SMAA.

This release has allowed us to develop a basic understanding of the problem domain and to indentify the difficulties involved in developing software for meta-analysis. Furthermore, it provides a starting point for discussions with users and domain experts. At present, however, we do not recommend that you use this software in your research, as the implementation is not yet mature.

Getting ADDIS

Download ADDIS version 0.2: addis-0.2.zip
Source code is also available: addis 0.2 source tag

Running ADDIS

Extract the addis-0.2.zip ZIP archive to a local directory. Then double-click the 'addis.jar' file. Now you should already have ADDIS 0.2 running with some provided example data. Verify this by comparing your screen with the screenshots below.

If double-clicking the JAR file did not launch ADDIS, you are most likely lacking the Java run-time environment. You can get this from java.com.
