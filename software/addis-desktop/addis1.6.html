Release history

Release 1.6.2

Release Date: 2011-06-10

ADDIS 1.6.2 is a bug fix release.

Release 1.6.1

Release Date: 2011-06-05

ADDIS v1.6.1 brings performance improvements and bug fixes to the network meta-analysis functionality. ADDIS v1.6 had difficulties performing large network meta-analyses that should now be solved.

Release 1.6

Release Date: 2011-04-15

We are happy to announce the release of ADDIS (Aggregate Data Drug Information System) version 1.6. New features include:

  • New formalized data model based on XML Schema, enabling future extensions of the data model.
  • Improvements to benefit-risk analysis.
  • The ability to take notes when inputting a study.
  • Added population baseline characteristics to example data.
  • Lots of bug fixes and usability improvements.

ADDIS v1.6 is mainly aimed to solidify what we already have, and to enable easier extension in the future. Thus, while it lacks major new features, we feel it is an important step forward.

Running ADDIS

Extract the addis-1.6.2.zip ZIP archive to a local directory. Then double-click the 'addis-1.6.2.jar' file. Now you should see the ADDIS welcome screen, where you can choose to load an example data set, load a file, or create a new data set. If you've never used ADDIS, we recommend that you take a look at the example data set.

If double-clicking the JAR file did not launch ADDIS, you are most likely lacking the Java run-time environment. You can get this from java.com.
